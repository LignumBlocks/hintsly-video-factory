# TICKET 302 — Generación de prompt_imagen

## Objetivo
Generar el prompt final de imagen a partir de la descripción visual.

## Descripción
- Crear función en `usecases/utils_prompt.py`.
- Combinar descripción visual con estilo cinematográfico.

## Criterios de aceptación
- Prompt consistente y limpio.
- No debe incluir instrucciones meta.
